# Experiment-12
## Aim of Experiment: Categorical Data Analysis using Python.
## Theory:
Operations Performed on the given datsaet:
1. Frequency Count:  
Counted occurrences of categorical variables using value_counts().  
2. Percentage Distribution:  
Calculated relative frequencies using normalize=True.  
3. Cross Tabulation:
Used pd.crosstab() to analyze relationships between.  
Gender vs Grade  
Department vs  Gender  
Category vs Payment Method  
4. Grouping:    
Applied groupby() to summarize data based on categories.  
5. Filtering:
Filtered dataset based on specific conditions.  
6. Sorting:                      
Sorted categorical data for better analysis.
7. Unique Values:  
Identifies unique categories using unique() and nunique.
## Functions / Commands Used:    
- pd.read_csv() → Load dataset.    
- head() → Display first few rows.  
- value_counts(normalize=True) → Percentage distribution.    
- pd.crosstab() → Cross-tabulation between variables.    
- groupby() → Group data based on categories.   
- size() → Count elements in each group.  
- count() → Count non-null values.  
- unique() → Get unique  values.  
- nunique() → Count unique values.  
- sort_values() → Sort data.  
- isin() → Filter specific category values.  
- loc[] → Filter rows based on conditions.
